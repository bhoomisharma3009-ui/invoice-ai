# Invoice Fraud Detection & Expense Analysis AI

## Overview
This project automates invoice processing using AI. It extracts invoice details, detects duplicate invoices, performs fraud analysis, and stores the results automatically.

## Features
- Upload PDF invoice
- Extract invoice details using AI
- Duplicate invoice detection
- Fraud score and risk level generation
- Google Sheets integration
- Structured JSON output

## Tech Stack
- n8n
- OpenAI
- Google Sheets
- OCR / PDF Text Extraction

## Workflow
1. Upload Invoice
2. Extract Text
3. AI extracts invoice details
4. Check for duplicate invoices
5. Fraud analysis
6. Save results to Google Sheets
